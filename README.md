# 🛡️ VladiMIR's Linux Server Automation (v2026)

Modular infrastructure management for NetCup, FastVDS, and VPN nodes.

## 🚀 Quick Start
1. git clone https://github.com/GinCz/Linux_Server_Public.git /root/scripts
2. Create /root/.server_env with your tokens
3. bash setup.sh

- 002: RU Node (Security)
