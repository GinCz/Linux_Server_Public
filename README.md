# VladiMIR's Linux Server Automation
Production-ready Bash scripts for secure Linux server setup.

## Structure
- System/: Core scripts, monitoring (infooo, sos)
- Security/: Global ban lists, sync scripts
- FastPanel/: Domain checkers, PHP limits
- Cloudflare/: Cloudflare proxy configs
- WordPress/: WAF rules
- VPN/: WireGuard and Samba setups
